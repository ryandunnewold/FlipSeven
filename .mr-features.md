# Flippy Keep Score

A score-keeping iOS app for the card game "Flip Seven." Players flip cards trying to get as close to 7 as possible without going over (busting).

## Implemented Features

### Game Management
- Create new games with custom player names
- Track multiple rounds of scoring
- Support for manual score entry (numeric input starting at 0, with confirm for bust)

### Scoring
- Per-round scoring sheet with manual entry
- Next Round button to advance (no auto-advance)
- Partial scoring: only enter scores for non-bust players; unscored players are auto-busted
- "Next Round" button shows count of players that will be auto-busted
- Running total scores per player

### Players
- Player avatars with customizable display
- Player score cards showing current standings
- Lifetime win/loss tracking: games played, wins, losses, round wins, and busts recorded per player
- Records stats view displays GP, W, L, RW, B columns

### UI/Navigation
- Tab-based navigation (Score, Players, Feed, Rules, New Game)
- Dark mode support
- Haptic feedback
- Custom app icon
- Feedback button in lobby and game header opens external feedback form in Safari

### App Store
- Fastlane integration for screenshots, builds, and uploads
- UI tests for automated screenshot capture across 3 iPhone device sizes
- App Store metadata (description, keywords, release notes)
- App branded as "Flippy Keep Score" (display name via CFBundleDisplayName, App Store name, description, keywords)
- Note: "Flip 7 Score Tracker" and "Flip 7 Scoreboard" names were taken by other apps
- Automated metadata setup via Spaceship API (copyright, categories, review contact, content rights)
- `setup_metadata` lane for configuring App Store Connect
- `deliver` lane for uploading metadata (creates new version if needed)
- `submit` lane for automated review submission
- `upload` lane for uploading pre-built IPAs to TestFlight
- Build uploaded to TestFlight (1.0.0, build #6)
- Build uploaded to TestFlight (1.1.0, build #4)
- Build uploaded to TestFlight (1.2.0, build #1)
- Screenshots uploaded to App Store Connect
- v1.1.0 metadata uploaded to App Store Connect via fastlane deliver
- ITSAppUsesNonExemptEncryption set in Info.plist (no manual encryption compliance needed)
- Manual export signing with "FlipSeven App Store" distribution profile (required for Xcode 26)
- Info.plist uses $(MARKETING_VERSION) and $(CURRENT_PROJECT_VERSION) build setting variables
